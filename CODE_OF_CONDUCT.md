# Code of Conduct

## Our Pledge
This repository contains educational teaching examples. We are committed to providing a welcoming, respectful, and harassment-free environment for anyone who wishes to learn from or contribute to this project.

## Our Standards
* **Be Welcoming:** Respect differing viewpoints, experience levels, and backgrounds.
* **Be Kind:** Gracefully accept constructive criticism and show empathy toward other community members.
* **Zero Tolerance:** Harassment, insults, trolling, or personal attacks will not be tolerated.

## Enforcement
Project maintainers have the right and responsibility to remove comments, issues, or code contributions that do not align with this Code of Conduct.
